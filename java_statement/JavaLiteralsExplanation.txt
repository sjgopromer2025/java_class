자바 리터럴

자바에서 리터럴(literal)은 변수에 직접 할당되는 상수 값을 의미합니다. 
이는 소스 코드에서 고정된 값을 나타냅니다. 아래는 자바에서 사용되는 리터럴의 종류입니다:

1. 정수 리터럴 (Integer Literals):
   - 정수를 나타냅니다.
   - 예: 10, -20, 0, 12345.

2. 부동소수점 리터럴 (Floating-Point Literals):
   - 소수를 나타냅니다.
   - 예: 3.14, -0.01, 2.0.

3. 문자 리터럴 (Character Literals):
   - 작은 따옴표(')로 묶인 단일 문자를 나타냅니다.
   - 예: 'a', 'Z', '1', '$'.

4. 문자열 리터럴 (String Literals):
   - 큰 따옴표(")로 묶인 문자들의 시퀀스를 나타냅니다.
   - 예: "Hello", "Java", "123".

5. 불리언 리터럴 (Boolean Literals):
   - true 또는 false 값을 나타냅니다.
   - 예: true, false.

6. 널 리터럴 (Null Literal):
   - null 값을 나타냅니다.
   - 예: null.

7. 이진수, 8진수, 16진수 리터럴 (Binary, Octal, and Hexadecimal Literals):
   - 이진수: 0b 또는 0B로 시작 (예: 0b1010).
   - 8진수: 0으로 시작 (예: 0123).
   - 16진수: 0x 또는 0X로 시작 (예: 0x1A).

자바 리터럴은 프로그램에서 상수 값을 정의하는 데 필수적입니다.
